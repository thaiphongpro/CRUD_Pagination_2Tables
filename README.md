# Employee Management API / API Quản Lý Nhân Viên

## Vietnamese / Tiếng Việt

Sử dụng cơ sở dữ liệu đính kèm gồm 3 bảng (Bảng Nhân Viên, Chức Vụ). Viết các API với yêu cầu:  

### 1. Hiển thị (Display API)
- Các trường cần hiển thị: **Mã nhân viên, Tên nhân viên, Giới tính, Tuổi, Tên chức vụ**.

### 2. Phân trang (Pagination API)
- Các trường cần hiển thị: **Mã nhân viên, Tên nhân viên, Giới tính, Tuổi, Tên chức vụ**.
- Số phần tử mỗi trang: **5 nhân viên/trang**.

### 3. Thêm / Sửa (Add / Update API)

### 4. Xóa (Delete API)
- Xóa theo **Mã nhân viên**.

### 5. Kiểm tra dữ liệu (Validate)
- Kiểm tra **không được để trống tất cả các trường** khi Thêm hoặc Sửa.

### 1. Display API
- Fields to display: **Employee Code, Employee Name, Gender, Age, Position Name**.

### 2. Pagination API
- Fields to display: **Employee Code, Employee Name, Gender, Age, Position Name**.
- Number of items per page: **5 employees per page**.

### 4. Delete API
- Delete by **Employee Code**.

### 5. Validation
